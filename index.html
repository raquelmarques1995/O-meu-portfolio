Projectos

Praias Portuguesas

Este projecto realizado em C, foi o primeiro projecto desenvolvido no CET de Tecnologias e Programação e teve como objetivo gerir dados sobre as praias portuguesas, organizados por localidade. A aplicação permite aos utilizadores adicionar, listar, modificar e pesquisar praias e localidades, além de exportar as informações para um ficheiro CSV.

Ferramentas e Tecnologias: Linha de Comandos, Code Block, C
Repositório GitHub

Vending Machine

Este projecto foi realizado no âmbito do módulo de Java e envolveu o desenvolvimento de uma aplicação que simula uma máquina de venda automática de snacks. A aplicação permite que os clientes comprem vários tipos de produtos e possibilita que a equipa gerencie a máquina, nomeadamente inserir produtos, eliminar e consultar o histórico.

Ferramentas e Tecnologias: IntelliJ, Java
Repositório GitHub

Gestão de Base de dados de Filmes

Este projecto foi desenvolvido no âmbito do módulo de Programação em SQL e contém um conjunto de queries de SQL desenvolvidas para trabalhar com a base de dados de filmes. O objetivo do projecto era criar uma série de queries com diferentes níveis de complexidade para extrair e analisar informação sobre filmes, realizadores e géneros na base de dados.

Ferramentas e Tecnologias: MySQL Workbench, SQL
Repositório GitHub

Gestão de Base de dados para Gestão de Projectos

Este projecto foi elaborado como projecto final do módulo de Criação de base de dados em SQL e tem como objetivo criar uma base de dados relacional para gerir projectos de uma empresa de TI. A base de dados inclui informações sobre clientes, projectos, tarefas e colaboradores e oferece recursos para registar, atualizar e monitorizar o progresso de projectos e tarefas.

Ferramentas e Tecnologias: MySQL Workbench, SQL

Aldeias Históricas Portuguesas
